# european-bank-churn-dashboard
Customer Segmentation &amp; Churn Pattern Analytics dashboard for European banking using Streamlit, featuring interactive visualizations, KPI metrics, and data-driven insights.
## 📊 Dashboard Preview

This dashboard provides insights into customer churn patterns across geography, age groups, and financial behavior.

![Customer Churn Dashboard](screenshots/dashboard.png)

### Key Insights
- Overall churn rate: **20.38%**
- Inactive customers have higher churn
- Customers aged 50+ are high risk
- Germany shows highest churn rate![dashboard](https://github.com/user-attachments/assets/e9e584b8-b2d6-4bd9-9215-06733cb414cf)
